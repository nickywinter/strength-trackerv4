# Strength Tracker

A simple offline HTML fitness tracker you can upload to GitHub Pages.

## Files
- `index.html` — the full app

## How to use with GitHub
1. Create a new GitHub repository.
2. Upload `index.html` to the root of the repo.
3. Commit the file.
4. In GitHub repo settings, enable GitHub Pages from the main branch/root.
5. Open the GitHub Pages URL on your iPhone and add it to your Home Screen.

## Notes
- Data is stored in your browser using localStorage.
- Backup/export is handled through the Backup Data box inside the app.
- If you switch browsers or clear site data, restore from your backup JSON.
